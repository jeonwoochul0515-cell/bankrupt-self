# 검색 노출·유입 실행 가이드

> 진단 결과: 사이트는 정상이고 SEO 기술 세팅도 거의 완비. 하지만 **검색엔진에 색인·노출이 안 되어** 방문자가 없음.
> 아래는 **사용자님이 직접 계정에 로그인해서 해야 하는** 작업들이다. 코드로는 대체 불가.

## 1순위 — 검색엔진에 사이트 등록 + sitemap 제출 (필수, 무료)

인증 메타태그는 `index.html`에 이미 심어져 있다(구글·네이버). 계정에서 인증만 누르면 된다.

### 구글 서치콘솔
1. https://search.google.com/search-console 접속 → 속성 추가 → "URL 접두어"에 `https://busan-hoiseng.pro` 입력
2. 소유권 확인 → "HTML 태그" 방식 선택 (이미 `google-site-verification` 메타 있음) → 확인
3. 좌측 메뉴 **Sitemaps** → `sitemap.xml` 입력 후 제출
4. 좌측 **URL 검사** → `https://busan-hoiseng.pro/` 입력 → "색인 생성 요청" 클릭
5. (며칠~2주 소요. "페이지" 보고서에서 색인 현황 확인)

### 네이버 서치어드바이저 (국내 핵심)
1. https://searchadvisor.naver.com 접속 → 웹마스터도구 → 사이트 등록 `https://busan-hoiseng.pro`
2. 소유확인 → "HTML 태그" (이미 `naver-site-verification` 메타 있음) → 확인
3. **요청 → 사이트맵 제출** → `sitemap.xml`
4. **요청 → 웹페이지 수집** → 메인 URL 직접 요청
5. **검증 → robots.txt** 확인 (이미 Yeti 봇 허용됨)

### 빙 웹마스터 (선택)
- https://www.bing.com/webmasters — 구글 서치콘솔 연동으로 한 번에 가져오기 가능

## 2순위 — 네이버 생태계 노출 (국내는 네이버가 검색 점유율 최상위)

네이버는 자사 콘텐츠(블로그·플레이스·카페)를 검색 상단에 우선 노출한다. 외부 사이트(.pro 도메인)만으로는 "부산 개인회생" 같은 경쟁 키워드 상위 노출이 매우 어렵다.

- **네이버 블로그 개설** → "부산 개인회생 변제금 후기", "2026 개인회생 생계비" 등 정보성 글 발행 → 본문에 `busan-hoiseng.pro` 링크
- **네이버 플레이스(스마트플레이스)** 등록 → 지역 검색("부산 개인회생") 노출. 단 사업자/주소 필요
- **지식iN 답변** → 관련 질문에 전문 답변 + 자가진단 링크

## 3순위 — 유료 검색광고 (즉시 유입, 비용 발생)

"부산 개인회생"은 법무법인·변호사들이 입찰하는 고비용 키워드다. 단기간 방문자 확보는 광고가 가장 빠르다.

- **네이버 검색광고(파워링크)** — 국내 유입 핵심. 키워드 입찰
- **구글 애즈** — 검색광고 + 디스플레이
- 광고 랜딩 시 UTM 파라미터를 붙이면 admin 대시보드에서 유입 채널 추적됨
  예) `https://busan-hoiseng.pro/?utm_source=naver&utm_medium=cpc&utm_campaign=busan_rehab`

## 4순위 — 신뢰도/백링크

- 카카오톡 채널, 인스타그램 등 SNS에 사이트 링크 게시 (OG 이미지 자동 노출됨)
- 관련 커뮤니티·디렉토리에 등록

## 효과 측정
- 모든 유입은 admin 대시보드(`/admin`)의 방문자 통계·유입 채널·전환 퍼널에서 확인
- 이번 추적 정상화 작업으로 쿠키 미동의 방문자도 방문수에 집계됨
- 더 정밀한 분석은 GTM(GTM-TZRMFF55) → GA4 연결 권장
